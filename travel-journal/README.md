# 🌍 旅途印记 - 旅行日记

一个精美的旅行日记网站，支持每日行程安排、随想记录、照片管理和高德地图地点标注。

## 功能特点

- **📅 每日行程安排** - 时间线展示每日行程计划
- **💭 旅途随想** - 记录每天的心情和感悟
- **📸 照片记录** - 拖拽上传、自动压缩、画廊浏览
- **🗺️ 高德地图集成** - 搜索地点、地图标注、按天分色
- **📤 导入导出** - JSON 格式备份，数据不丢失
- **📱 响应式设计** - 手机、平板、电脑完美适配
- **💾 本地存储** - 数据保存在浏览器，隐私安全

## 使用方法

### 在线访问
直接访问 GitHub Pages 地址即可使用。

### 本地运行
```bash
cd travel-journal
python3 -m http.server 8000
# 访问 http://localhost:8000
```

## 操作指南

1. 点击「✈️ 新建旅程」创建旅行
2. 点击「+ 添加一天」记录每日行程
3. 填写行程安排、随想、上传照片
4. 搜索地点进行地图标注
5. 切换「地图总览」查看所有行程地点
6. 切换「相册」查看所有照片
7. 定期使用「📤 导出」备份数据

## 技术栈

- 纯前端 HTML/CSS/JS，无需后端
- 高德地图 JS API 2.0
- LocalStorage 本地存储
- 照片自动压缩，节省存储空间
