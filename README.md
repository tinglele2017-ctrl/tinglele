# 成长记忆 - 宝贝成长网络相册

一个温馨的孩子成长记忆相册网站，支持按年龄整理照片，时间树展示，带有温情动画效果。

## 功能特点

✅ **按年龄分类** - 预设 0-18 岁年龄节点，每个年龄单独管理照片
✅ **时间树展示** - 水平时间轴直观展示不同年龄段，丝滑动效
✅ **照片上传** - 支持点击上传、拖拽上传，可同时上传多张
✅ **图片查看** - 点击放大查看原图，支持描述记录
✅ **本地存储** - 照片保存在浏览器本地，不会上传到服务器，隐私安全
✅ **温情动画** - 浮动爱心背景，渐入动画，柔和交互动效
✅ **响应式设计** - 支持手机、平板、电脑访问
✅ **无需后端** - 纯前端 HTML/CSS/JS，开箱即用

## 部署方法

### 方式一：直接本地打开（简单测试）
直接用浏览器打开 `index.html` 即可使用。

### 方式二：本地启动HTTP服务器
```bash
# 在项目目录下
python3 -m http.server 8000
# 然后访问 http://你的服务器IP:8000 即可
```

### 方式三：部署到公共网络（推荐）
可以部署到以下免费静态网站托管服务：

1. **GitHub Pages**（免费）
   - 上传这三个文件到 GitHub 仓库
   - 开启 GitHub Pages
   - 即可获得公开访问地址

2. **Vercel**（免费）
   - 导入 GitHub 仓库
   - 自动部署，获得域名

3. **Netlify**（免费）
   - 拖放文件夹到 Netlify
   - 立即上线

4. **Cloudflare Pages**（免费）
   - 连接 GitHub，自动部署

## 使用说明

1. 打开网站后，左侧时间轴选择年龄（0-18岁）
2. 点击右下角 `+` 按钮添加照片
3. 选择照片文件，填写描述（可选），点击上传
4. 照片会自动保存在浏览器本地存储中
5. 点击照片可以放大查看
6. 鼠标悬停在照片上可以点击删除

## 技术说明

- 纯前端实现，无需数据库
- 使用 LocalStorage 存储照片数据
- 照片会被转换为 base64 存储，所有数据都在本地
- 如果清除浏览器数据，照片会丢失，重要照片请做好备份
